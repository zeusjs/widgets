btnAsync
directive in module zeus.widgets

Description

A directive that provides a loading spinner to buttons

Usage

as class
<ANY class="btn-async [indicate-loading: {boolean};]">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
indicateLoading
(optional)
boolean

A two-way bound boolean that controls whether or not the spinner is shown
